Methodology & Architecture

This application leverages deep learning techniques on resting-state fMRI data to identify early biomarkers of Parkinson's Disease.

Data Acquisition

Data is sourced from the Parkinson's Progression Markers Initiative (PPMI). The dataset consists of 4D resting-state fMRI scans preprocessed using the fMRIPrep pipeline.

Feature Engineering

We extract Functional Connectivity (FC) matrices using the Nilearn library. This transforms the 4D time-series data into a correlation matrix representing brain region interactions.
